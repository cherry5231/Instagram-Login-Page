# Instagram Login Page 

A beginner frontend project inspired by the Instagram login page interface.

## About The Project

This project was created while learning:

* HTML
* CSS
* Frontend layout design
* Flexbox
* Styling components

The goal of this project was to practice recreating a real-world website UI instead of only building basic tutorial projects.

## Features

* Instagram-inspired login layout
* Flexbox-based page structure
* Styled login inputs
* Facebook login button
* Responsive-style centered layout
* Hover effects
* Custom buttons and containers
* Image integration

## Technologies Used

* HTML5
* CSS3

## What I Learned

While building this project, I practiced:

* Structuring webpages using HTML
* Using CSS classes
* Styling buttons and forms
* Using Flexbox for layouts
* Creating reusable UI sections
* Working with images and spacing
* Building cleaner frontend designs
